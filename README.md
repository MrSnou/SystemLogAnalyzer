#  System Log Analyzer (Work in progress)

**System Log Analyzer** is desktop app programmed in JavaFX, for processing and analyzing Windows logs.
Project was made for learning purposes to learn how to program graphic applications and gaining more skill in building apps in **MVC** pattern.

---

##  Description

In Application you will be able to view, filter and analyze Windows Logs in saved .log files on your hard drive.
App will detect any errors, allow you to search for exact errors which occur and allow you to print report about system process.


In future project will be expanded with;
- handling logs from various sources,
- statistical analysis of errors,
- export results to CSV or JSON files.

---

##  Tech stack 

- **Java 25**
- **JavaFX** – graphic interface
- **Maven** – dependency management
- **Spring Boot (planned)** – potential backend extension in subsequent stages
- **MVC (Model-View-Controller)** – project architecture
